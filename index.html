1. Apa itu version control

Version Control adalah sistem yang digunakan untuk mengelola perubahan pada file atau kode program.

2. Perbedaan Git dan GitHub

Git adalah sebuah sistem version control yang digunakan untuk melacak perubahan pada kode program di komputer lokal. Git memungkinkan developer menyimpan riwayat perubahan dan mengelola versi kode.

GitHub adalah platform berbasis web yang digunakan untuk menyimpan repository Git secara online. GitHub memudahkan developer untuk berkolaborasi, berbagi kode, dan mengelola proyek secara bersama-sama.

3. Apa itu HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML digunakan untuk menampilkan teks, gambar, link, tabel, dan berbagai elemen lainnya pada website.
